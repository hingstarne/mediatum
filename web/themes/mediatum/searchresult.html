Allgemeine Tipps zur Suche:

  1. Probieren Sie verschiedene Suchbegriffe aus. Überlegen Sie sich Synonyme für die von Ihnen verwendeten Suchanfragen.
    Verwenden Sie weniger Stichwörter auf einmal.
  2. Mit Begr* können Sie nach allen Begriffen suchen, die mit Begr beginnen. Suchbegriffe dürfen nicht mit einem Stern (*) beginnen und werden nach dem ersten Stern abgeschnitten.
  3. Benutzen Sie den Browsingbaum auf der linken Seite, um nach Dokumenten und Bildern zu browsen, statt einer speziellen Suche.
  4. Vermeiden Sie als Suchbegriffe Bindewörter (und, oder) und bestimmte Artikel (der, die, das).
  5. Wenn Sie eine Auswahl aus einer Auswahlliste getroffen haben, die durch eine Zahl (z.B.: foobar (9)) gekennzeichnet ist, so kann es sein, daß Sie trotzdem keine Resultate erhalten, weil Sie nicht die Berechtigung haben, all diese Objekte zu sehen, oder weil die Objekte noch nicht eingeordnet sind.
  6. Probieren Sie eine Volltextsuche. Manche spezielle Suchbegriffe tauchen in den Metadaten bzw. in der Zusammenfassung noch nicht auf, sondern erst im Dokument selber.

Search Tips:

  1. Try different search queries. Try synonyms. Reduce the number of words in your query.
  2. Use Wor* to search for all words that start with Wor. Search terms may not begin with a star (*) and are cut off after the first star.
  3. Use the browing tree on the left side to browse for documents and images, instead of using a search.
  4. Don't use common words (like and, or, the, a, of, etc.)
  5. If you selected a value from a dropdown list, marked by an occurence number (e.g. foobar (9)), you might still not receive any queries for it if you don't have the necessary permissions to view those objects.
  6. Try a fulltext search. Some search words don't occur in the metadata, only in the document itself.
